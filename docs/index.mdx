Kunai is a terminal-first media shell: search, resolve direct-provider streams locally, supervise mpv, and recover when catalogs drift. This documentation states what works today, what is beta-only, and how to debug with evidence.

## Recommended reading order

<Steps>
<Step>
### Install and verify
Read [Getting started](/docs/users/getting-started), then [Platforms](/docs/users/platforms) for OS-specific tools. Binary installs via `install.sh` / `install.ps1` are beta supported - see [Install and update](/docs/users/install-and-update).
</Step>
<Step>
### Learn the shell
Use [What you can do](/docs/users/what-you-can-do) for the capability map, the [Feature tour](/docs/users/feature-tour) for live command/provider tables, and [Commands and shortcuts](/docs/users/commands-and-shortcuts) for daily palette work.
</Step>
<Step>
### Plan for failure
[Playback and recovery](/docs/users/playback-and-recovery) and [Providers](/docs/users/providers) explain recover vs fallback. [Troubleshooting](/docs/users/troubleshooting) indexes common symptoms. [Diagnostics](/docs/users/diagnostics-and-reporting) shows how to export evidence safely.
</Step>
<Step>
### Tune and extend
[Customization](/docs/users/customization) covers config, mpv, and Discord presence. Developers: [Contribute](/docs/developer/contribute).
</Step>
</Steps>
