3 · Why this won't become slop

• One accent per theme + a 3-color state set (green/red + neutral ramp) — never more.
• Type-hue is contained to Stats; lists stay neutral so they never go rainbow.
• The paint-mix heatmap drives brightness by minutes and only hue by ratio, with capped saturation — so it reads as intensity first, identity second, and mixed days look intentional, not muddy.
• Every theme is tuned to read on its base color alone; glow and shadow are browser sugar that the terminal drops without loss.
