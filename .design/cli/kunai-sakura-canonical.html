Kunai · ❀ Sakura — canonical

The single visual authority. This supersedes kunai-sakura.html (good but narrow) and demotes playback-postplay/workbench-rec-v1.html to structural ideas only — its amber/teal palette is dead. Every surface here is Sakura two-note (rose = act, mint = ready/done, crimson = real error, plum = series-complete only), uses the full terminal width, and carries a deliberate weight ramp so it never reads flat.

What this fixes vs shipped code: post-play regained its episode-page rail + structured actions + rec cards; rails widened to earn 178 cols; titles brightened and the single primary action now dominates; tracks/recovery promoted to first-class. Green notes under each surface say exactly what changed and which file owns it.
