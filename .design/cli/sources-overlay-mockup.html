Phase timeline

User presses Play
    │
    ├─ Phase A (blocking) ─────────────────────────────
    │    1. Resolve Luffy only (60–90s max)
    │    2. fail? → Zoro → Nami → Sanji (ordered, not parallel)
    │    3. success → start mpv immediately
    │
    └─ Phase B (lazy, after “playing”) ───────────────
         4. Probe Zoro, Sanji, Brook, … (short timeout each)
         5. Update Sources list: ✓ available / ✕ failed / … probing
         6. User can switch without full provider fallback if already probed
